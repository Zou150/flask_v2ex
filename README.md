## 安装说明

### clone项目到本地或服务器
```python
git clone https://github.com/guoweikuang/flask_v2ex.git
```
### 安装依赖包
```
pip install -r requirements.txt
```
### 初始化数据库
```python
python manage.py db init
python manage.py db migrate
python manage.py db upgrade
```
### 启动程序
```
python manage.py runserver 
```
